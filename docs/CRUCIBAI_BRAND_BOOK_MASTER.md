## VISUAL DESIGN SYSTEM

### TYPOGRAPHY FRAMEWORK

**Inspired by:** Microsoft Edge design patterns

**Font Family:** Segoe UI (primary), system fallbacks

**Typography Scale:**

| Element | Size | Weight | Line Height | Letter Spacing | Usage |
|---------|------|--------|-------------|-----------------|-------|
| **Hero Headline** | 56px | 700 | 1.2 | 0.5px | Main page titles |
| **Section Headline** | 48px | 700 | 1.3 | 0.3px | Major sections |
| **Subheading** | 32-40px | 600 | 1.4 | 0.2px | Section introductions |
| **Body Text** | 16-18px | 400 | 1.6 | 0px | Main content |
| **Caption/Label** | 14-16px | 500 | 1.5 | 0.1px | Feature labels, captions |
| **Small Text** | 12-14px | 400 | 1.5 | 0px | Metadata, timestamps |

**Hierarchy Principles:**
- Generous line height (1.4-1.6) for readability
- Bold headlines (700-800 weight) for visual impact
- Breathing room between elements
- Professional, accessible appearance

### IMAGE PLACEMENT STRATEGY

**Inspired by:** Microsoft Edge design patterns

**Layout Pattern 1: Text Left + Image Right (50/50 Split)**
- Headline + body text on left (dark background)
- Large, high-quality image on right (rounded corners)
- Image is the visual anchor
- Creates balance and visual interest
- **Use Cases:** Hero section, feature introductions, use case demonstrations

**Layout Pattern 2: Text Left + Feature Grid (Right)**
- Headline + body on left
- 3x3 or 4x3 grid of feature cards on right
- Each card has icon/image + label
- Organized, scannable
- **Use Cases:** 115 agents showcase, feature overview, capability matrix

**Layout Pattern 3: Split Screen (Full Width)**
- Two large images side-by-side
- Minimal text overlay
- Images are the hero
- **Use Cases:** Competitive comparison, before/after, evolution story

**Layout Pattern 4: Text Left + Large Image Right**
- Headline + description + CTA button
- Large, immersive image
- Image tells the story
- **Use Cases:** Value propositions, use cases, benefits

**Layout Pattern 5: Text Left + UI Mockup (Right)**
- Shows product in context
- Real-world use case
- Builds trust
- **Use Cases:** Product demonstrations, workflow examples

**Layout Pattern 6: Full-Width Hero Image**
- Large, immersive image
- Minimal text overlay
- Emotional connection
- **Use Cases:** Page headers, emotional storytelling, vision statements

---

### IMAGE STYLING GUIDELINES

- **Border Radius:** 12-16px (rounded corners)
- **Shadows:** Subtle drop shadow (0 10px 30px rgba(0,0,0,0.3))
- **Contrast:** High contrast between image and background
- **Quality:** Professional photography or high-quality generated images
- **Aspect Ratios:** 16:9 (hero), 1:1 (grid), 4:3 (featured)
- **Text on Images:** Dark text on light backgrounds, light text on dark backgrounds (high contrast)
